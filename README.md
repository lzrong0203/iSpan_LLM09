## 💡 你會學到什麼？

### 🧠 核心概念（用簡單的方式理解）
- **LLM 是什麼？** → 想像成一個「超級自動完成」功能
- **Token 是什麼？** → AI 理解文字的「積木」
- **Transformer 是什麼？** → 讓 AI 理解上下文的「大腦」
- **RAG 是什麼？** → 讓 AI「查書」而不是「背書」
- **Ollama 是什麼？** → 在自己電腦運行 AI 的工具

### 🛠️ 實作技能
1. **基礎技能**
   - 設置 Python 環境
   - 使用 Google Colab
   - 執行範例程式

2. **核心技能**
   - 建立聊天機器人
   - 調用 OpenAI API
   - 實作 RAG 系統
   - 優化 Prompt

3. **進階技能**
   - 理解模型微調
   - 使用 LoRA 技術
   - 部署應用程式
   - Ollama 本地部署

## 📦 環境需求

### 最低配置
- **作業系統：** Windows 10/11, macOS 10.15+, Ubuntu 20.04+
- **Python：** 3.8 或更高版本
- **記憶體：** 8GB RAM
- **硬碟：** 20GB 可用空間
- **網路：** 穩定的網路連線

### 建議配置
- **記憶體：** 16GB RAM 以上
- **GPU：** NVIDIA GPU with 6GB+ VRAM（選配）
- **硬碟：** 50GB 可用空間

> 💡 **沒有 GPU？沒關係！**  
> 可以使用 Google Colab 免費 GPU 或調用 API
